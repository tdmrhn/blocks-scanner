Easily scan and list the Gutenberg blocks used on your site. Quickly edit or view the posts that use the blocks.

== Description ==

Blocks Scanner is a WordPress plugin that allows you to easily scan and list the Gutenberg blocks used on your site. With Blocks Scanner, you can quickly edit or view the posts that use specific blocks, streamlining your editing workflow.

== Features ==

* Scan and list Gutenberg blocks used on your WordPress site.
* Quickly edit or view the posts that contain specific blocks.
* Streamline your editing workflow by easily identifying and managing blocks.

== Usage ==

* Once activated, navigate to Tools > Blocks Scanner in your WordPress admin panel.
* The plugin will automatically scan all posts, pages, and custom post types.
* You can then view the list of posts that utilize blocks on your site and take appropriate actions quickly.

== Frequently Asked Questions ==

= How do I scan Gutenberg blocks on my site? =

After activating the Blocks Scanner plugin, navigate to Tools > Blocks Scanner in your WordPress admin panel. The plugin will automatically scan all posts, pages, and custom post types to list the Gutenberg blocks used on your site.

= Can I edit or view posts containing specific blocks? =

Yes, Blocks Scanner allows you to quickly edit or view the posts that contain specific blocks. Once you've scanned your site, you can easily identify and manage blocks from the plugin's interface.

= What if I encounter any issues or have feature requests? =

If you encounter any issues, have feature requests, or need support, please visit the [GitHub repository](https://github.com/tdmrhn/blocks-scanner) and create an issue. Our team will be happy to assist you.

= What about compatibility with other plugins/themes? =

Blocks Scanner is designed to work smoothly with most WordPress themes and plugins. However, if you experience any compatibility issues, please let us know by creating an issue on the GitHub repository.

= How often does Blocks Scanner update its block list? =

Blocks Scanner updates its block list in real-time as you scan your site. This ensures that you always have the most up-to-date information about the Gutenberg blocks used on your WordPress site.

= Is Blocks Scanner compatible with multisite installations? =

Yes, Blocks Scanner is compatible with multisite installations of WordPress. You can activate the plugin network-wide and use it to scan and manage Gutenberg blocks across all sites in your network.
